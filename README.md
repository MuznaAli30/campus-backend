# Campus Backend 🏫💻

## 📌 Description
Backend for **Campus Management System** built with **Node.js and Express.js**.  
Handles student, staff, and campus data with API routes and MongoDB integration.  

## 🚀 Features
- RESTful APIs for campus operations ⚡  
- CRUD operations for students and staff 📝  
- MongoDB database integration 💾  

## 🛠 Tech Stack
- Node.js, Express.js, MongoDB, JavaScript 💻  

## ⚙️ Installation
1. Clone repo: `git clone https://github.com/MuznaAli30/campus-backend.git`  
2. Install dependencies: `npm install`  
3. Start server: `node index.js`  
Server runs at http://localhost:5000 🌐
